Project 基本資訊

實作內容

  1. 幣別DB維護功能（查詢 / 新增 / 修改 / 刪除 功能API。）。
    • 查詢（GET）：http://localhost:8080/api/search/{要查詢的字串}
    • 新增（POST）：http://localhost:8080/api/，在 body 中放入 json 格式資料
    • 修改（PUT）：http://localhost:8080/api/，在 body 中放入 json 格式資料
    • 刪除（DELETE）：http://localhost:8080/api/
  2. 呼叫 coindesk 的API。
  3. 呼叫 coindesk 的API，並進行資料轉換，組成新API。 此新API 提供：
    • A. 更新時間（時間格式範例：1990/01/01 00:00:00）。
    • B. 幣別相關資訊（幣別，幣別中文名稱，以及匯率）。
  4. 單元測試

開發框架與工具

  • 框架： Spring boot
  • package 庫： maven
  • 資料庫： H2
  • IDE： eclipse
  • 匯率資料來源： coindesk api

資料庫建立指令


DROP TABLE IF EXISTS currency;

CREATE TABLE currency (
  id INT AUTO_INCREMENT  PRIMARY KEY,
  en_name VARCHAR(50) NOT NULL,
  ch_name VARCHAR(50) NOT NULL
);

INSERT INTO currency (en_name, ch_name) VALUES
  ('USD', '美元'),
  ('GBP', '英鎊'),
  ('EUR', '歐元');
